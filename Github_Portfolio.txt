Delete File Locally via Git (Command Line)

1. Clone your repository (if not already cloned):
	git clone https://github.com/sunil-chakraborty/sunil-c.github.io.git
	cd sunil-c.github.io

2. issue dos command (for file del dir ren copy etc.)

3. Stage and commit the change:
	git add .
	git commit -m "Deleted unnecessary file"
	git push origin main

Replace main with master if your default branch is master.
